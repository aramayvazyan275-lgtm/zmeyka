# Yandex Game

Этот проект представляет собой шаблон HTML-игры, соответствующий требованиям платформы Яндекс.Игры.

## Структура проекта

- `yandex_game.html` - Основной HTML-файл игры с необходимыми метатегами и стилями
- `manifest.json` - Файл манифеста для поддержки Progressive Web App
- `README.md` - Данный файл с описанием

## Требования Яндекс.Игр

Проект включает в себя все необходимые элементы для соответствия требованиям Яндекс.Игр:

1. **Метатег подтверждения** - `<meta name="yandex-game" content="true">`
2. **Адаптивный дизайн** - Использование viewport и стилей для корректного отображения на разных устройствах
3. **Поддержка темной темы** - Черный фон и соответствующие настройки
4. **Отключение нежелательных действий** - Отключение выделения текста, вызова контекстного меню и других действий
5. **Поддержка SDK** - Код для инициализации и использования Yandex Games SDK
6. **PWA возможности** - Файл manifest.json для поддержки Progressive Web App

## Использование SDK

В коде реализована инициализация Yandex Games SDK и показаны примеры использования различных API:

- Хранилище (Storage)
- Таблица лидеров (Leaderboard)
- Реклама (Ads)
- Платежи (Payments)

Для активации этих функций раскомментируйте соответствующие участки кода.

## Запуск

Для запуска игры в локальной среде используйте локальный веб-сервер, так как SDK может требовать HTTPS для корректной работы.
